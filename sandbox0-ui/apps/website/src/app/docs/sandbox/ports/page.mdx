# Port Exposure

Expose sandbox ports to make services accessible via public URLs. This allows external access to web servers, APIs, and other services running inside a sandbox.

## Exposed Port Model

| Field | Type | Description |
|-------|------|-------------|
| `port` | integer | Port number exposed inside the sandbox |
| `resume` | boolean | Auto-resume sandbox when traffic hits this port |
| `public_url` | string | Public URL for accessing the exposed port (read-only) |

<Callout variant="info">
When <code>resume</code> is enabled, a paused sandbox will automatically resume when receiving traffic on the exposed port.
</Callout>

---

## Get Exposed Ports

List all exposed ports for a sandbox.

<Endpoint method="GET">
/api/v1/sandboxes/{'{id}'}/exposed-ports
</Endpoint>

### Response

| Field | Type | Description |
|-------|------|-------------|
| `sandbox_id` | string | Sandbox ID |
| `exposed_ports` | array | List of exposed port configurations |
| `exposure_domain` | string | Base domain for public URLs (e.g., `aws-us-east-1.sandbox0.app`) |

}
fmt.Printf("Exposure domain: %s\\n", ports.ExposureDomain)
for _, p := range ports.Ports {
    fmt.Printf("Port %d: %s\\n", p.Port, p.PublicURL)
}`
    },

## Expose a Port

Expose a port to make it publicly accessible.

<Endpoint method="PUT">
/api/v1/sandboxes/{'{id}'}/exposed-ports
</Endpoint>

### Request Body

| Field | Type | Description |
|-------|------|-------------|
| `ports` | array | List of port configurations to expose |

<Tabs
  tabs={[
    {
      label: "Go",

}
fmt.Printf("Exposure domain: %s\\n", resp.ExposureDomain)
for _, p := range resp.Ports {
    fmt.Printf("Port %d -> %s\\n", p.Port, p.PublicURL)

## Expose Multiple Ports

Expose multiple ports at once.

<Tabs
  tabs={[
    {
      label: "Go",
      language: "go",
      code: `// Expose multiple ports

    {Port: 3000, Resume: false},
    {Port: 9000, Resume: true},
})
if err != nil {
    log.Fatal(err)
}

    fmt.Printf("Port %d -> %s (resume: %v)\\n", p.Port, p.PublicURL, p.Resume)
}`
    },
    {
      label: "Python",
      language: "python",
      code: `# Expose multiple ports
resp = sandbox.expose_ports([
    {"port": 8080, "resume": True},
    {"port": 3000, "resume": False},
    {"port": 9000, "resume": True},
])
for p in resp.ports:
    print(f"Port {p.port} -> {p.public_url} (resume: {p.resume})")`
    },
    {
      label: "TypeScript",
      language: "typescript",
      code: `// Expose multiple ports
const resp = await sandbox.exposePorts([
    { port: 8080, resume: true },
    { port: 3000, resume: false },
    { port: 9000, resume: true },
]);
for (const p of resp.ports) {
    console.log(\`Port \${p.port} -> \${p.publicUrl} (resume: \${p.resume})\`);

      language: "bash",

## Complete Example: Web Server

Start a web server and expose it publicly.

<Tabs
  tabs={[
    {
      label: "Go",
      language: "go",
      code: `// Start a web server (async, don't wait for completion)
result, err := sandbox.Cmd(ctx,
    "python3 -m http.server 8080 --directory /tmp",
    sandbox0.WithCmdWait(false),
)
if err != nil {
    log.Fatal(err)
}
fmt.Printf("Server started, context: %s\\n", result.ContextID)

// Wait for server to start
time.Sleep(2 * time.Second)

// Expose port 8080
portsResp, err := sandbox.ExposePort(ctx, 8080, true)
if err != nil {
    log.Fatal(err)
}

fmt.Printf("Server accessible at:\\n")
for _, p := range portsResp.Ports {
    fmt.Printf("  %s\\n", p.PublicURL)
}

// Verify the server is working
resp, err := sandbox.Cmd(ctx, "curl -s http://localhost:8080/")
if err != nil {
    log.Fatal(err)
}
fmt.Printf("Local response: %s\\n", resp.OutputRaw)`
    },
    {
      label: "Python",
      language: "python",
      code: `import time

# Start a web server (async)
result = sandbox.cmd(
    "python3 -m http.server 8080 --directory /tmp",
    wait=False,
)
print(f"Server started, context: {result.context_id}")

# Wait for server to start
time.sleep(2)

# Expose port 8080
ports_resp = sandbox.expose_port(8080, resume=True)

print("Server accessible at:")
for p in ports_resp.ports:
    print(f"  {p.public_url}")

# Verify the server is working
resp = sandbox.cmd("curl -s http://localhost:8080/")
print(f"Local response: {resp.output_raw}")`
    },
    {
      label: "TypeScript",
      language: "typescript",
      code: `// Start a web server (async)
const result = await sandbox.cmd(
    'python3 -m http.server 8080 --directory /tmp',
    { wait: false },
);
console.log('Server started, context:', result.contextId);

// Wait for server to start
await new Promise((r) => setTimeout(r, 2000));

// Expose port 8080

console.log('Server accessible at:');
for (const p of portsResp.ports) {

console.log('Local response:', resp.outputRaw);`

      label: "CLI",

# Expose port 8080

# Port 8080 exposed at:
# https://sb-abc123-8080.aws-us-east-1.sandbox0.app

## Auto-Resume Feature

When a sandbox is paused with `resume: true` on an exposed port, incoming traffic will automatically resume the sandbox.

<Tabs
  tabs={[
    {
      label: "Go",
      language: "go",
      code: `// Expose port with auto-resume
_, err := sandbox.ExposePort(ctx, 8080, true)

// Pause the sandbox
_, err = client.PauseSandbox(ctx, sandbox.ID)

// Incoming HTTP request to the public URL will automatically
// resume the sandbox before routing the request`
    },
    {
      label: "Python",
      language: "python",
      code: `# Expose port with auto-resume
sandbox.expose_port(8080, resume=True)

# Pause the sandbox
client.pause_sandbox(sandbox.id)

# Incoming HTTP request to the public URL will automatically
# resume the sandbox before routing the request`
    },
    {
      label: "TypeScript",
      language: "typescript",
      code: `// Expose port with auto-resume
await sandbox.exposePort(8080, true);

// Pause the sandbox
await client.pauseSandbox(sandbox.id);

// Incoming HTTP request to the public URL will automatically
// resume the sandbox before routing the request`
    },
    {
      label: "CLI",
      language: "bash",
      code: `# Expose port with auto-resume
s0 ports expose sb_abc123 8080 --resume

# Pause the sandbox
s0 sandbox pause sb_abc123

# Incoming HTTP request to the public URL will automatically
# resume the sandbox before routing the request`
    }
  ]}
/>

<Callout variant="info">
Auto-resume adds a small delay to the first request after pausing (typically 2-5 seconds) while the sandbox resumes. Subsequent requests are served immediately.
</Callout>
