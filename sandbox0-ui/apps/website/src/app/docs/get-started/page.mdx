# Get Started

Sandbox0 provides isolated execution environments for AI Agents with persistent storage, session state maintenance, sub-200ms cold starts, and easy self-hosted deployment.

## What is Sandbox0?

Sandbox0 is an enterprise-grade AI Agent Sandbox platform that enables you to:

- **Run code securely** in isolated environments with full process control
- **Persist data** across sandbox lifecycles with volume storage and snapshots
- **Control network access** with fine-grained policies
- **Scale efficiently** with template-based sandbox pools and fast cold starts

### Control Plane vs Data Plane

Sandbox0 follows a control plane / data plane architecture:

| Layer | Components | Purpose |
|-------|------------|---------|
| **Control Plane** | edge-gateway, scheduler | Multi-cluster routing, tenant management, request forwarding |
| **Data Plane** | internal-gateway, manager, storage-proxy, netd | Sandbox lifecycle, process execution, file operations, storage |

Each provider/region (e.g., `aws/us-east-1`) has an independent data plane and control plane deployment with dedicated PostgreSQL and S3 storage.

### Claim Your First Sandbox

Create a sandbox from the default template:

<Tabs
  tabs={[
    {
      label: "Go",
      language: "go",
      code: `package main

import (
    "context"
    "fmt"
    "log"
    "os"

    "github.com/sandbox0-ai/sdk-go"
)

func main() {
    client, err := sandbox0.NewClient(
        sandbox0.WithToken(os.Getenv("SANDBOX0_TOKEN")),
    )
    if err != nil {
        log.Fatal(err)
    }

    ctx := context.Background()

    // Claim a sandbox from the "default" template
    sandbox, err := client.ClaimSandbox(ctx, "default",
        sandbox0.WithSandboxTTL(300), // 5 minutes TTL
    )
    if err != nil {
        log.Fatal(err)
    }
    fmt.Printf("Sandbox ID: %s\\n", sandbox.ID)
    fmt.Printf("Status: %s\\n", sandbox.Status)

    // Clean up when done
    defer client.DeleteSandbox(ctx, sandbox.ID)
}`,
      mode: "run",
      needs: "infra",
      session: "get-started-go",
      env: ["SANDBOX0_TOKEN"]
    },
    {
      label: "Python",
      language: "python",
      code: `from sandbox0 import Client
import os

client = Client(token=os.environ["SANDBOX0_TOKEN"])

# Claim a sandbox from the "default" template
sandbox = client.claim_sandbox(
    template="default",
    config={"ttl": 300},  # 5 minutes TTL
)
print(f"Sandbox ID: {sandbox.id}")
print(f"Status: {sandbox.status}")

# Clean up when done
client.delete_sandbox(sandbox.id)`,
      mode: "run",
      needs: "infra",
      session: "get-started-python",
      env: ["SANDBOX0_TOKEN"]
    },
    {
      label: "JavaScript",
      language: "typescript",
      code: `import { Sandbox0 } from '@sandbox0/sdk';

const client = new Sandbox0({ token: process.env.SANDBOX0_TOKEN });

// Claim a sandbox from the "default" template
const sandbox = await client.claimSandbox({
    template: 'default',
    config: { ttl: 300 },  // 5 minutes TTL
});
console.log('Sandbox ID:', sandbox.id);
console.log('Status:', sandbox.status);

// Clean up when done
await client.deleteSandbox(sandbox.id);`,
      mode: "run",
      needs: "infra",
      session: "get-started-typescript",
      env: ["SANDBOX0_TOKEN"]
    },
    {
      label: "CLI",
