# Files

Manage files and directories within a sandbox. The Files API provides full POSIX file operations including read, write, delete, move, and real-time file watching.

## File Model

### FileInfo

| Field | Type | Description |
|-------|------|-------------|
| `name` | string | File name |
| `path` | string | Full path |
| `type` | string | File type: `file`, `dir`, `symlink` |
| `size` | integer | File size in bytes |
| `mode` | string | Unix file mode (e.g., `0755`) |
| `mod_time` | string | Last modification time (ISO 8601) |
| `is_link` | boolean | Whether it's a symlink |
| `link_target` | string | Symlink target path (if applicable) |

## Watch Files

Watch a directory for file system changes in real-time via WebSocket.

<Endpoint method="GET">
/api/v1/sandboxes/{'{id}'}/files/watch
</Endpoint>

### WebSocket Protocol

**Client Messages:**

| Action | Description | Example |
|--------|-------------|---------|
| `subscribe` | Subscribe to a path | `{"action": "subscribe", "path": "/tmp", "recursive": true}` |
| `unsubscribe` | Unsubscribe from a watch | `{"action": "unsubscribe", "watch_id": "watch-123"}` |

**Server Messages:**

| Type | Description | Example |
|------|-------------|---------|
| `subscribed` | Subscription confirmed | `{"type": "subscribed", "watch_id": "watch-123", "path": "/tmp"}` |
| `event` | File event occurred | `{"type": "event", "watch_id": "watch-123", "event": "write", "path": "/tmp/a.txt"}` |
| `unsubscribed` | Unsubscription confirmed | `{"type": "unsubscribed", "watch_id": "watch-123"}` |
| `error` | Error occurred | `{"type": "error", "error": "permission denied"}` |

### File Events

| Event | Description |
|-------|-------------|
| `create` | File or directory created |
| `write` | File content modified |
| `rename` | File or directory renamed |
| `chmod` | File permissions changed |
| `remove` | File or directory deleted |

<Tabs
  tabs={[
    {
      label: "Go",
      language: "go",
      code: `// Watch files for changes
watchCtx, cancel := context.WithTimeout(ctx, 30*time.Second)
defer cancel()

events, errs, unsubscribe, err := sandbox.WatchFiles(watchCtx, "/workspace", true)
if err != nil {
    log.Fatal(err)
}
defer unsubscribe()

// Process events
for {
    select {
    case ev, ok := <-events:
        if !ok {
            return
        }
        fmt.Printf("Event: %s on %s\\n", ev.Event, ev.Path)
    case err, ok := <-errs:
        if ok && err != nil {
            log.Printf("Watch error: %v", err)
        }
    case <-watchCtx.Done():
        fmt.Println("Watch timeout")
        return
    }
}`
    },
    {
      label: "Python",
      language: "python",
      code: `# Watch files for changes
import time

events, errors, unsubscribe = sandbox.watch_files("/workspace", recursive=True)

try:
    for event in events:
        print(f"Event: {event.event} on {event.path}")
        if event.event == "write":
            print(f"File modified: {event.path}")
except KeyboardInterrupt:
    pass
finally:
    unsubscribe()`
    },
    {
      label: "TypeScript",
      language: "typescript",
      code: `// Watch files for changes
const { events, errors, unsubscribe } = await sandbox.watchFiles('/workspace', true);

events.on('data', (event) => {
    console.log(\`Event: \${event.event} on \${event.path}\`);
    if (event.event === 'write') {
        console.log(\`File modified: \${event.path}\`);
    }
});

errors.on('data', (error) => {
    console.error('Watch error:', error);
});

// Unsubscribe after 30 seconds
setTimeout(() => {
    unsubscribe();
}, 30000);`
    },
    {
      label: "CLI",
      language: "bash",
      code: `# Watch files for changes
s0 file watch sb_abc123 /workspace --recursive

# Output (live):
# [create] /workspace/newfile.txt
# [write] /workspace/newfile.txt
# [rename] /workspace/newfile.txt -> /workspace/renamed.txt
# [remove] /workspace/renamed.txt`
    }
  ]}
/>
