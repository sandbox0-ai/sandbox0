# Source Priority

This file defines the source priority contract for the Sandbox0 skill
framework.

1. Source code
2. `pkg/apispec/openapi.yaml`
3. `s0` CLI help and implementation
4. Bundled docs included in the released skill artifact
5. Hosted website docs

Bundled docs are release-matched reference material. They must not override
source code, OpenAPI, or CLI behavior.
