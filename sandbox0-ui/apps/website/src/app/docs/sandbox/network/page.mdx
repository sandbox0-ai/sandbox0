# Network Policy

Control network access for sandboxes. Network policies allow you to restrict outbound traffic, block specific domains, or allow only certain endpoints.

## Network Modes

| Mode | Description | Use Case |
|------|-------------|----------|
| `allow-all` | Allow all outbound traffic | Default, unrestricted access |
| `block-all` | Block all outbound traffic | Maximum security, air-gapped environments |

## Egress Policy

When using `block-all` mode, you can define exceptions via egress rules:

| Field | Type | Description |
|-------|------|-------------|
| `allowed_domains` | array | Whitelist of allowed domains |
| `denied_domains` | array | Blacklist of blocked domains |
| `allowed_cidrs` | array | Whitelist of allowed IP CIDRs |
| `denied_cidrs` | array | Blacklist of blocked IP CIDRs |
| `allowed_ports` | array | Whitelist of allowed destination ports |
| `denied_ports` | array | Blacklist of blocked destination ports |

<Callout variant="info">
Egress rules are only effective when combined with <code>block-all</code> mode. In <code>allow-all</code> mode, all traffic is permitted regardless of rules.
</Callout>

## Get Network Policy

Retrieve the current network policy for a sandbox.

<Endpoint method="GET">
/api/v1/sandboxes/{'{id}'}/network
</Endpoint>

<Tabs
  tabs={[
    {
      label: "Go",
      language: "go",
      code: `// Get current network policy
policy, err := sandbox.GetNetworkPolicy(ctx)
if err != nil {
    log.Fatal(err)
}
fmt.Printf("Mode: %s\\n", policy.Mode)
if egress, ok := policy.Egress.Get(); ok {
    fmt.Printf("Allowed domains: %v\\n", egress.AllowedDomains)
}`
    },
    {
      label: "Python",
      language: "python",
      code: `# Get current network policy
from sandbox0.apispec.types import UNSET

policy = sandbox.get_network_policy()
print(f"Mode: {policy.mode}")
if policy.egress is not UNSET:
    print(f"Allowed domains: {policy.egress.allowed_domains}")`
    },
    {
      label: "TypeScript",
      language: "typescript",
      code: `// Get current network policy
const policy = await sandbox.getNetworkPolicy();
console.log('Mode:', policy.mode);
if (policy.egress) {
    console.log('Allowed domains:', policy.egress.allowedDomains);
}`
    },
    {
      label: "CLI",
      language: "bash",
      code: `s0 sandbox network get -s sb_abc123`
    }
  ]}
/>

## Update Network Policy

Update the network policy for a sandbox.

<Endpoint method="PUT">
/api/v1/sandboxes/{'{id}'}/network
</Endpoint>

### Request Body

| Field | Type | Description |
|-------|------|-------------|
| `mode` | string | Network mode: `allow-all` or `block-all` |
| `egress` | object | Egress policy rules (optional) |

---

### Allow All Traffic

Allow all outbound network access (default behavior).

<Tabs
  tabs={[
    {
      label: "Go",
      language: "go",
      code: `// Allow all traffic
_, err = sandbox.UpdateNetworkPolicy(ctx, apispec.TplSandboxNetworkPolicy{
    Mode: apispec.TplSandboxNetworkPolicyModeAllowAll,
})
if err != nil {
    log.Fatal(err)
}
fmt.Println("Network policy updated: allow-all")`
    },
    {
      label: "Python",
      language: "python",
      code: `# Allow all traffic
from sandbox0.apispec.models.tpl_sandbox_network_policy import TplSandboxNetworkPolicy
from sandbox0.apispec.models.tpl_sandbox_network_policy_mode import TplSandboxNetworkPolicyMode

sandbox.update_network_policy(TplSandboxNetworkPolicy(mode=TplSandboxNetworkPolicyMode.ALLOW_ALL))
print("Network policy updated: allow-all")`
    },
    {
      label: "TypeScript",
      language: "typescript",
      code: `// Allow all traffic
await sandbox.updateNetworkPolicy({ mode: 'allow-all' });
console.log('Network policy updated: allow-all');`
    },
    {
      label: "CLI",
      language: "bash",
      code: `s0 sandbox network update -s sb_abc123 --mode allow-all`
    }
  ]}
/>

### Allow Specific Domains

Block all traffic except for specific allowed domains.

<Tabs
  tabs={[
    {
      label: "Go",
      language: "go",
      code: `// Block all except specific domains
_, err = sandbox.UpdateNetworkPolicy(ctx, apispec.TplSandboxNetworkPolicy{
    Mode: apispec.TplSandboxNetworkPolicyModeBlockAll,
    Egress: apispec.NewOptNetworkEgressPolicy(apispec.NetworkEgressPolicy{
        AllowedDomains: []string{"github.com", "pypi.org", "api.openai.com"},
    }),
})
if err != nil {
    log.Fatal(err)
}
fmt.Println("Network policy updated: only github.com, pypi.org, api.openai.com allowed")`
    },
    {
      label: "Python",
      language: "python",
      code: `# Block all except specific domains
from sandbox0.apispec.models.network_egress_policy import NetworkEgressPolicy

sandbox.update_network_policy(
    TplSandboxNetworkPolicy(
        mode=TplSandboxNetworkPolicyMode.BLOCK_ALL,
        egress=NetworkEgressPolicy(
            allowed_domains=["github.com", "pypi.org", "api.openai.com"],
        ),
    )
)
print("Network policy updated: only allowed domains")`
    },
    {
      label: "TypeScript",
      language: "typescript",
      code: `// Block all except specific domains
await sandbox.updateNetworkPolicy({
    mode: 'block-all',
    egress: {
        allowedDomains: ['github.com', 'pypi.org', 'api.openai.com'],
    },
});
console.log('Network policy updated: only allowed domains');`
    },
    {
      label: "CLI",

## Set Network Policy at Creation

Configure network policy when claiming a sandbox.

<Tabs
  tabs={[
    {
      label: "Go",
      language: "go",
      code: `// Claim sandbox with network policy
sandbox, err = client.ClaimSandbox(ctx, "default",
    sandbox0.WithSandboxHardTTL(600),
    sandbox0.WithSandboxNetworkPolicy(apispec.TplSandboxNetworkPolicy{
        Mode: apispec.TplSandboxNetworkPolicyModeAllowAll,
    }),
)
if err != nil {
    log.Fatal(err)
}`
    },
    {
      label: "Python",
      language: "python",
      code: `# Claim sandbox with network policy
from sandbox0.apispec.models.sandbox_config import SandboxConfig

sandbox = client.sandboxes.claim(
    template="default",
    config=SandboxConfig(
        hard_ttl=600,
        network=TplSandboxNetworkPolicy(mode=TplSandboxNetworkPolicyMode.ALLOW_ALL),
    ),
)`
    },
    {
      label: "TypeScript",
      language: "typescript",
      code: `// Claim sandbox with network policy
const newSandbox = await client.sandboxes.claim('default', {
    hardTtl: 600,
    network: { mode: 'allow-all' },
});`

    {
      label: "CLI",

## Test Network Connectivity

Verify network policy by making requests from the sandbox.

<Tabs
  tabs={[
    {
      label: "Go",
      language: "go",
      code: `// Test network connectivity
const shell = \`/bin/curl -s -o /dev/null -w "%{http_code}\\n" --max-time 3 https://github.com\`
resp, err := sandbox.Cmd(ctx, shell)
if err != nil {
    log.Fatal(err)
}
fmt.Printf("GitHub response before blocking: %s\\n", resp.OutputRaw)

// Block all traffic
_, err = sandbox.UpdateNetworkPolicy(ctx, apispec.TplSandboxNetworkPolicy{
    Mode: apispec.TplSandboxNetworkPolicyModeBlockAll,
})
if err != nil {
    log.Fatal(err)
}

// Test again (should fail)
resp, err = sandbox.Cmd(ctx, shell)
if err != nil {
    fmt.Println("Request blocked as expected")
}
fmt.Printf("GitHub response after blocking: %s\\n", resp.OutputRaw)`
    },
    {
      label: "Python",
      language: "python",
      code: `# Test network connectivity
shell = '/bin/curl -s -o /dev/null -w "%{http_code}\\n" --max-time 3 https://github.com'
resp = sandbox.cmd(shell)
print(f"GitHub response before blocking: {resp.output_raw.strip()}")

# Block all traffic
sandbox.update_network_policy(TplSandboxNetworkPolicy(mode=TplSandboxNetworkPolicyMode.BLOCK_ALL))

# Test again (should fail)
try:
    resp = sandbox.cmd(shell)
    print(f"GitHub response after blocking: {resp.output_raw.strip()}")
except Exception as e:
    print("Request blocked as expected")`
    },
    {
      label: "TypeScript",
      language: "typescript",
      code: `// Test network connectivity
const shell = '/bin/curl -s -o /dev/null -w "%{http_code}\\n" --max-time 3 https://github.com';
let resp = await newSandbox.cmd(shell);
console.log('GitHub response before blocking:', resp.outputRaw.trim());

// Block all traffic
await newSandbox.updateNetworkPolicy({ mode: 'block-all' });

// Test again (should fail)
try {
    resp = await newSandbox.cmd(shell);
    console.log('Github response after blocking:', resp.outputRaw.trim());
} catch (e) {
    console.log('Request blocked as expected');
}`
