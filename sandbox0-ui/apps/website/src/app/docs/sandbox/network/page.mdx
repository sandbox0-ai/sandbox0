# Network Policy

Control network access for sandboxes. Network policies allow you to restrict outbound traffic, block specific domains, or allow only certain endpoints.

## Network Modes

| Mode | Description | Use Case |
|------|-------------|----------|
| `allow-all` | Allow all outbound traffic | Default, unrestricted access |
| `block-all` | Block all outbound traffic | Maximum security, air-gapped environments |

## Egress Policy

When using `block-all` mode, you can define exceptions via egress rules:

| Field | Type | Description |
|-------|------|-------------|
| `allowed_domains` | array | Whitelist of allowed domains |
| `blocked_domains` | array | Blacklist of blocked domains |
| `allowed_cidrs` | array | Whitelist of allowed IP CIDRs |
| `blocked_cidrs` | array | Blacklist of blocked IP CIDRs |
| `allowed_ports` | array | Whitelist of allowed destination ports |
| `blocked_ports` | array | Blacklist of blocked destination ports |

<Callout variant="info">
Egress rules are only effective when combined with <code>block-all</code> mode. In <code>allow-all</code> mode, all traffic is permitted regardless of rules.
</Callout>

## Get Network Policy

Retrieve the current network policy for a sandbox.

<Endpoint method="GET">
/api/v1/sandboxes/{'{id}'}/network
</Endpoint>

<Tabs
  tabs={[
    {
      label: "Go",

if err != nil {

if egress, ok := policy.Egress.Get(); ok {
    fmt.Printf("Allowed domains: %v\\n", egress.AllowedDomains)
}`
    },
    {

    },

}`

## Update Network Policy

Update the network policy for a sandbox.

<Endpoint method="PUT">
/api/v1/sandboxes/{'{id}'}/network
</Endpoint>

### Request Body

| Field | Type | Description |
|-------|------|-------------|
| `mode` | string | Network mode: `allow-all` or `block-all` |
| `egress` | object | Egress policy rules (optional) |

## Allow Specific Domains

Block all traffic except for specific allowed domains.

<Tabs
  tabs={[
    {
      label: "Go",
      language: "go",
      code: `// Block all except specific domains
_, err := sandbox.UpdateNetworkPolicy(ctx, apispec.TplSandboxNetworkPolicy{
    Mode: apispec.TplSandboxNetworkPolicyModeBlockAll,
    Egress: apispec.NewOptNetworkEgressPolicy(apispec.NetworkEgressPolicy{
        AllowedDomains: []string{"github.com", "pypi.org", "api.openai.com"},
    }),
})
if err != nil {
    log.Fatal(err)
}
fmt.Println("Network policy updated: only github.com, pypi.org, api.openai.com allowed")`
    },
    {
      label: "Python",
      language: "python",
      code: `# Block all except specific domains
sandbox.update_network_policy(
    mode="block-all",
    egress={
        "allowed_domains": ["github.com", "pypi.org", "api.openai.com"],
    },
)
print("Network policy updated: only allowed domains")`
    },
    {
      label: "TypeScript",
      language: "typescript",
      code: `// Block all except specific domains
await sandbox.updateNetworkPolicy({
    mode: 'block-all',
    egress: {
        allowedDomains: ['github.com', 'pypi.org', 'api.openai.com'],

## Test Network Connectivity

Verify network policy by making requests from the sandbox.

  tabs={[

    log.Fatal(err)

    fmt.Println("Request blocked as expected")
}`

    {

## Complete Example

A complete example showing network policy management.

<Tabs
  tabs={[
    {
      label: "Go",
      language: "go",
      code: `// Get current policy
current, err := sandbox.GetNetworkPolicy(ctx)
if err != nil {
    log.Fatal(err)
}
fmt.Printf("Current policy: %+v\\n", current)

// Test network connectivity
const shell = \`/bin/curl -s -o /dev/null -w "%{http_code}\\n" --max-time 3 https://github.com\`
resp, err := sandbox.Cmd(ctx, shell)
if err != nil {
    log.Fatal(err)
}
fmt.Printf("GitHub HTTP code: %s", resp.OutputRaw)

// Block all traffic
_, err = sandbox.UpdateNetworkPolicy(ctx, apispec.TplSandboxNetworkPolicy{
    Mode: apispec.TplSandboxNetworkPolicyModeBlockAll,
})
if err != nil {
    log.Fatal(err)
}

// Allow specific domains only
_, err = sandbox.UpdateNetworkPolicy(ctx, apispec.TplSandboxNetworkPolicy{
    Mode: apispec.TplSandboxNetworkPolicyModeBlockAll,
    Egress: apispec.NewOptNetworkEgressPolicy(apispec.NetworkEgressPolicy{
        AllowedDomains: []string{"github.com"},
    }),
})
if err != nil {
    log.Fatal(err)
}
fmt.Println("Only github.com is now allowed")`
    },
    {
      label: "Python",
      language: "python",
      code: `# Get current policy
current = sandbox.get_network_policy()
print(f"Current policy: {current.mode}")

// Block all traffic
