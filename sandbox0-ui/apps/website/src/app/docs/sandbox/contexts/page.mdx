# Contexts (Process Runtime)

Contexts are execution units within a sandbox. Each context runs a process and can be either a **REPL** (stateful) or **Cmd** (stateless) type.

## Context Types

| Type | Description | Use Case |
|------|-------------|----------|
| **REPL** | Stateful, interactive process | Code evaluation, interactive shells, persistent sessions |
| **Cmd** | Stateless, one-shot command | Single commands, scripts, non-interactive execution |

### REPL vs Cmd

<Callout variant="info">
**REPL contexts preserve state** between calls - variables, environment, and working directory persist. **Cmd contexts are stateless** - each execution starts fresh.
</Callout>

```python
# REPL: State preserved between calls
>>> x = 2
>>> print(x)
2

# Cmd: Each command is independent
$ x=2
$ echo $x    # Empty - variable not preserved
```

## Create Context

Create a new context in a sandbox.

<Endpoint method="POST">
/api/v1/sandboxes/{'{id}'}/contexts
</Endpoint>

### Request Body

| Field | Type | Description |
|-------|------|-------------|
| `type` | string | Context type: `repl` or `cmd` |
| `repl` | object | REPL configuration (required if type=repl) |
| `cmd` | object | Command configuration (required if type=cmd) |
| `cwd` | string | Working directory |
| `env_vars` | object | Environment variables |
| `pty_size` | object | Terminal size `{rows, cols}` |
| `ttl_sec` | integer | Context time-to-live in seconds |
| `idle_timeout_sec` | integer | Idle timeout in seconds |
| `wait_until_done` | boolean | Wait for completion (default: false) |

### REPL Configuration

| Field | Type | Description |
|-------|------|-------------|
| `language` | string | REPL language (python, bash, node, etc.) |
| `input` | string | Initial input to send |

### Cmd Configuration

| Field | Type | Description |
|-------|------|-------------|
| `command` | array | Command and arguments |

## High-Level: Run and Cmd

For common use cases, use the convenience methods `Run` (REPL) and `Cmd` (command).

### Run (REPL)

Execute code in a REPL context. State is preserved between calls.

<Tabs
  tabs={[
    {
      label: "Go",
      language: "go",
      code: `// First call - set variable

    log.Fatal(err)
}
fmt.Print(result.OutputRaw)

// Second call - variable preserved
result, err = sandbox.Run(ctx, "python", \`print(x)\`)
if err != nil {
    log.Fatal(err)
}
fmt.Print(result.OutputRaw)  // Output: 2`
    },
    {
      label: "Python",

print(result.output_raw)

# Second call - variable preserved

console.log(result.outputRaw);

### Cmd (One-shot Command)

Execute a stateless command.

<Tabs
  tabs={[
    {
      label: "Go",
      language: "go",
      code: `// One-shot command - stateless
result, err := sandbox.Cmd(ctx, \`/bin/sh -c "echo hello"\`)
if err != nil {
    log.Fatal(err)
}
fmt.Print(result.OutputRaw)  // Output: hello

// Variables NOT preserved between Cmd calls
sandbox.Cmd(ctx, \`/bin/sh -c "x=3"\`)
result, _ = sandbox.Cmd(ctx, \`/bin/sh -c "echo \\$x"\`)
fmt.Print(result.OutputRaw)  // Output: (empty)`
    },
    {
      label: "Python",
      language: "python",
      code: `# One-shot command - stateless
result = sandbox.cmd('/bin/sh -c "echo hello"')
print(result.output_raw)  # Output: hello

# Variables NOT preserved between cmd calls
sandbox.cmd('/bin/sh -c "x=3"')
result = sandbox.cmd('/bin/sh -c "echo $x"')
print(result.output_raw)  # Output: (empty)`
    },
    {
      label: "TypeScript",
      language: "typescript",
      code: `// One-shot command - stateless
let result = await sandbox.cmd('/bin/sh -c "echo hello"');
console.log(result.outputRaw);  // Output: hello

// Variables NOT preserved between cmd calls
await sandbox.cmd('/bin/sh -c "x=3"');
result = await sandbox.cmd('/bin/sh -c "echo $x"');
console.log(result.outputRaw);  // Output: (empty)`
    },
    {
      label: "CLI",
      language: "bash",
      code: `s0 sandbox exec sb_abc123 -- /bin/sh -c "echo hello"
# Output: hello`
    }
  ]}
/>

## WebSocket Streaming

For real-time I/O, connect to a context via WebSocket.

<Endpoint method="GET">
/api/v1/sandboxes/{'{id}'}/contexts/{'{ctx_id}'}/ws
</Endpoint>

### WebSocket Protocol

**Client Messages:**

| Type | Description | Example |
|------|-------------|---------|
| `input` | Send input to stdin | `{"type": "input", "data": "ls\n", "request_id": "req-1"}` |
| `resize` | Resize terminal | `{"type": "resize", "rows": 24, "cols": 80}` |
| `signal` | Send signal | `{"type": "signal", "signal": "INT"}` |

**Server Messages:**

| Type | Description | Example |
|------|-------------|---------|
| `output` | stdout/stderr output | `{"type": "output", "source": "stdout", "data": "hello\n"}` |
| `done` | Request completed | `{"type": "done", "request_id": "req-1"}` |

<Tabs
  tabs={[
    {
      label: "Go",
      language: "go",
      code: `// Create context first
ctxResp, err := sandbox.CreateContext(ctx, apispec.CreateContextRequest{
    Type: apispec.NewOptProcessType(apispec.ProcessTypeRepl),
    Repl: apispec.NewOptCreateREPLContextRequest(apispec.CreateREPLContextRequest{
        Language: apispec.NewOptString("python"),
    }),
})

// Connect WebSocket
conn, _, err := sandbox.ConnectWSContext(ctx, ctxResp.ID)
if err != nil {
    return err
}
defer conn.Close()

// Send input
msg := map[string]any{
    "type":       "input",
    "data":       "print('hello')\\n",
    "request_id": "req-1",
}
conn.WriteJSON(msg)

// Read output
for {
    _, message, err := conn.ReadMessage()
    if err != nil {
        break
    }
    var msg struct {
        Type   string \`json:"type"\`
        Source string \`json:"source"\`
        Data   string \`json:"data"\`
    }
    json.Unmarshal(message, &msg)
    if msg.Type == "output" {
        fmt.Print(msg.Data)
    }
}`
    },
    {
      label: "Python",
      language: "python",
      code: `import websocket
import json

# Connect to WebSocket
ws = sandbox.connect_websocket("ctx_xyz789")

# Send input
ws.send(json.dumps({
    "type": "input",
    "data": "print('hello')\\n",
    "request_id": "req-1",
}))

# Read output
for message in ws:
    msg = json.loads(message)
    if msg.get("type") == "output":
        print(msg["data"], end="")`
    },
    {
      label: "TypeScript",
      language: "typescript",
      code: `// Connect to WebSocket

// Send input
ws.send(JSON.stringify({
    type: 'input',
    data: "print('hello')\\n",

}));

// Read output

  const msg = JSON.parse(event.data);
  if (msg.type === 'output') {
    console.log(msg.data);
  }
};`
    },

      label: "CLI",
